# FindMyOwner

QR-based item ownership and recovery platform.

## Project Structure

```text
findmyowner/
├── backend/
│   ├── app/
│   └── tests/
└── frontend/
    ├── public/
    └── src/
```

## Planned Stack

- Backend: FastAPI, SQLAlchemy, Alembic, PostgreSQL
- Frontend: React, Vite, TypeScript, Tailwind CSS
- Auth: Firebase Authentication
- Notifications: Twilio SMS, optional email
- Storage: Amazon S3
- Infrastructure: Docker, AWS
